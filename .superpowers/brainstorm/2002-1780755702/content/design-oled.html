对齐 OLED SSD1306 数据格式 — OLED_ShowImage()

OLED 图像数据格式 (SSD1306)

数据排布: 128×64 单色
col 0col 1col 2...col 127
Page 0 (row 0-7) byte 0 byte 1 byte 2 ... byte 127
Page 1 (row 8-15) byte 128 byte 129 ...... byte 255
...
Page 7 (row 56-63) byte 896 byte 897 ...... byte 1023

每个 bit 控制一个像素：1=亮, 0=暗 | LSB 在顶部
数据顺序: Image[page * 128 + col]

对齐现有工程格式 — Diode[] 风格

STM32 侧播放代码（极简）

// 逐帧播放视频
for (int f = 0; f < VIDEO_FRAME_COUNT; f++) {
    OLED_ShowImage(0, 0, 128, 64, video_frames[f]);
    OLED_Update();
    Delay_ms(1000 / VIDEO_FPS);
}
